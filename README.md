## Info ###
> Lo esencial sería utilizar MergeSort, que tiene complejidad O(nLog(n)). Se supone que esta hecho, y tiene que probarse con listas alternativas para que funcione, por lo tanto he utilizado LinkedList y ArrayList (Aprovechando las ventajas de cada una. [Mira aquí](http://stackoverflow.com/questions/322715/when-to-use-linkedlist-over-arraylist)) en donde se lo necesita. Sin embargo no podemos utilizar esos tipos, por lo que estaba implementando nuestra propia lista [DoubleLinkedList](https://bitbucket.org/dnarvaez27/doublelinkedlist), aunque no ha funcionado tan bien como con las API de Java. Igual trabajemos en ellas a ver como nos sale :grinning:
